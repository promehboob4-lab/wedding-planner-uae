🎯 This guide covers 2 paths: (A) Deploy this HTML file in 2 minutes, or (B) Build the full Next.js production app. Start with A to show clients, then build B for the real platform.

Option A — Deploy HTML File ⚡ 2 Minutes

Perfect for demos, investor pitches, and client presentations. Deploy this exact file live instantly.

1

Download This File

Save this HTML file as index.html to your computer.

2

Create GitHub Account & Repository

Go to github.com → Sign up free → Click "New Repository" → Name it nikah-hub → Click "Create Repository".

💡 GitHub is where your code lives. Vercel connects directly to GitHub to deploy automatically.
3

Upload File to GitHub

On your new repository page, click "uploading an existing file" → Drag your index.html → Click "Commit changes".

4

Connect to Vercel

Go to vercel.com → "Sign up with GitHub" → "Add New Project" → Select your nikah-hub repo → Framework: Other → Click Deploy.

🎉 That's it! In 30 seconds you'll have a live URL like nikah-hub.vercel.app
5

Add Custom Domain (Optional)

In Vercel: Settings → Domains → Type nikahhub.ae → Follow DNS instructions. Buy domain from Namecheap (~AED 50/year). Vercel provides free SSL automatically.

Option B — Full Production Next.js App 🏗️ Recommended

Build the real production platform with database, auth, payments, and AI. Use this for actual launch.

1

Install Node.js & Create Project

First install Node.js from nodejs.org. Then open Terminal (Mac) or Command Prompt (Windows):

# Install Node.js first from nodejs.org node --version # Should show v18 or higher # Create the Next.js project npx create-next-app@latest nikah-hub \ --typescript \ --tailwind \ --app \ --src-dir # When prompted, answer: Would you like to use ESLint? → Yes Would you like to customize import alias? → No cd nikah-hub
2

Install All Required Packages

# Core dependencies npm install @supabase/supabase-js @supabase/auth-helpers-nextjs npm install stripe @stripe/stripe-js npm install @anthropic-ai/sdk npm install resend npm install cloudinary npm install date-fns npm install react-hot-toast npm install framer-motion npm install react-datepicker npm install zustand # State management
3

Project Folder Structure

nikah-hub/ ├── src/ │ ├── app/ │ │ ├── page.tsx # Landing page │ │ ├── dashboard/page.tsx # Couple dashboard │ │ ├── vendors/page.tsx # Vendor listings │ │ ├── vendor-portal/page.tsx # Vendor dashboard │ │ ├── booking/[id]/page.tsx # Booking flow │ │ ├── invite/[id]/page.tsx # Digital invite │ │ └── api/ │ │ ├── ai-planner/route.ts # Claude AI API │ │ ├── bookings/route.ts # CRUD bookings │ │ ├── payments/route.ts # Stripe webhooks │ │ ├── vendors/route.ts # Vendor CRUD │ │ ├── guests/route.ts # Guest manager │ │ └── contracts/route.ts # PDF generation │ ├── components/ │ │ ├── ChatWidget.tsx │ │ ├── VendorCard.tsx │ │ ├── BudgetSlider.tsx │ │ ├── BookingModal.tsx │ │ ├── SeatingChart.tsx │ │ ├── GuestTable.tsx │ │ ├── InviteBuilder.tsx │ │ └── CountdownTimer.tsx │ └── lib/ │ ├── supabase.ts │ ├── stripe.ts │ ├── claude.ts │ └── utils.ts ├── .env.local # Your secret keys └── package.json
4

Setup Supabase Database (Free)

Go to supabase.com → Create free account → New Project → Name: "nikah-hub" → Copy your API keys. Then run this SQL in Supabase SQL Editor:

-- Users table create table users ( id uuid primary key default uuid_generate_v4(), email text unique not null, name text, role text default 'couple', -- couple | vendor | admin wedding_date date, budget integer, guest_count integer, culture text, theme text, created_at timestamptz default now() ); -- Vendors table create table vendors ( id uuid primary key default uuid_generate_v4(), user_id uuid references users(id), business_name text not null, category text not null, description text, price_min integer, price_max integer, location text, rating decimal(3,2) default 0, review_count integer default 0, verified boolean default false, availability jsonb, created_at timestamptz default now() ); -- Bookings table create table bookings ( id uuid primary key default uuid_generate_v4(), user_id uuid references users(id), vendor_id uuid references vendors(id), event_date date, status text default 'pending', deposit_amount integer, total_amount integer, contract_url text, created_at timestamptz default now() ); -- Guests table create table guests ( id uuid primary key default uuid_generate_v4(), user_id uuid references users(id), name text not null, relation text, rsvp_status text default 'pending', table_number integer, dietary text, phone text, email text ); -- Reviews table create table reviews ( id uuid primary key default uuid_generate_v4(), booking_id uuid references bookings(id), user_id uuid references users(id), vendor_id uuid references vendors(id), rating integer check (rating between 1 and 5), comment text, ai_verified boolean default false, created_at timestamptz default now() );
5

Create Environment Variables File

Create a file called .env.local in your project root:

# Supabase (from supabase.com → Project Settings → API) NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI... SUPABASE_SERVICE_ROLE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI... # Stripe (from stripe.com → Developers → API Keys) NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_... STRIPE_SECRET_KEY=sk_live_... STRIPE_WEBHOOK_SECRET=whsec_... # Claude AI (from console.anthropic.com) ANTHROPIC_API_KEY=sk-ant-... # Resend Email (from resend.com) RESEND_API_KEY=re_... # Cloudinary (from cloudinary.com) NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your-cloud-name CLOUDINARY_API_KEY=123456789 CLOUDINARY_API_SECRET=abc123... # App URL NEXT_PUBLIC_APP_URL=https://nikahhub.ae
⚠️ NEVER commit .env.local to GitHub. Add it to .gitignore immediately!
6

Build the AI Planner API Route

Create src/app/api/ai-planner/route.ts:

import Anthropic from '@anthropic-ai/sdk'; import { NextRequest, NextResponse } from 'next/server'; const client = new Anthropic(); export async function POST(req: NextRequest) { const { messages, budget, guests, culture } = await req.json(); const response = await client.messages.create({ model: 'claude-sonnet-4-6', max_tokens: 1024, system: `You are Laila, an expert AI wedding planner for Dubai. You know Dubai vendors, pricing, seasons, and cultural customs. Current user: Budget ${budget} AED, ${guests} guests, ${culture} wedding. Always respond in the same language the user writes in. Be warm, specific, and actionable. Include AED prices.`, messages: messages, }); return NextResponse.json({ reply: response.content[0].text }); }

# Initialize git in your project git init git add . git commit -m "🎊 Initial NikahHub commit" # Create repo on github.com, then: git remote add origin https://github.com/YOUR_USERNAME/nikah-hub.git git branch -M main git push -u origin main

# Option 1: Vercel CLI (fastest) npm install -g vercel vercel login vercel # Follow prompts, choose your GitHub repo # Option 2: Vercel Dashboard 1. Go to vercel.com → Login with GitHub 2. Click "Add New Project" 3. Import your nikah-hub repo 4. Add all .env.local variables in "Environment Variables" section 5. Click Deploy → Done in ~60 seconds! ✅
🎉 Every time you push to GitHub, Vercel auto-deploys! No manual work needed ever again.
